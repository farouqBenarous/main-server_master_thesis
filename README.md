# How to Run


* export the spring profile local or demo 
``` 
export SPRING_PROFILES_ACTIVE=local 
```


* Compile the project using maven 
``` 
mvn clean install  
```

* Run the database 
``` 
Check docker folder for details 
```

* configure a spring runner in Intilij
